## Key Source File Responsibilities

### `src/recomp/rt.h`

- `M68KCtx` struct: `D[8]`, `A[8]`, `N Z V C X S I`
- Memory macros: `MR8/16/32(addr)`, `MW8/16/32(addr, val)` — call `rt_read*/rt_write*`
- Flag macros: `RT_SET_NZ_8/16/32`, `RT_ADD_FLAGS_*`, `RT_SUB_FLAGS_*`
- Condition code predicates: `RT_CC_EQ`, `RT_CC_NE`, `RT_CC_MI`, etc.
- `GameFnEntry` struct: `{uint32_t addr; GameFn fn}`
- `RT_UNIMPL(msg)` — prints to stderr + `abort()`
- `extern uint8_t *g_mem` (8 MB flat)
- `rt_init(binary, load_addr, stack_top)`, `rt_fini()`

### `src/recomp/rt.c`

- `g_mem`: 8 MB `calloc`'d buffer
- `rt_init()`: loads chip RAM dump file at `load_addr`
- `is_hw(addr)`: true for `$BFD000–$BFDFFF`, `$BFE000–$BFEFFF`, `$DFF000–$DFFFFF`
- `rt_read*/rt_write*`: route to `hw_read*/hw_write*` or direct `g_mem` access
- `dispatch_lookup()`: binary search over `g_fn_table[]`
- `rt_call()` / `rt_jump()`: call via dispatch table, abort if not found
- `extern const GameFnEntry g_fn_table[]` and `extern const int g_fn_count` — defined in `game.c`

### `src/recomp/hw.h`

Public API only — see `hw.c` for implementation.

### `src/recomp/hw.c`

- `s_regs[512]`: OCS register shadow array indexed by `(offset >> 1)`
- Custom chip write handlers: DMACON, INTENA, BPLCON0, BPLxPTH/L, COLOR00-31, DIWSTRT, COP1LCH/L
- Custom chip read handlers: DMACONR, VPOSR (synthetic), VHPOSR, JOY0DAT
- CIA-A: fire button via `s_joy_buttons`; CIA-B: timer simulation
- `hw_present_frame()`: SDL event pump → `hw_render_frame()` → SDL_RenderPresent → vsync
- `hw_render_frame()`: 6-bitplane OCS decode into `s_fb[320×256]`
- `hw_load_disk()`: `fread` from WHDLoad disk image into `g_mem`
- `hw_advance_scanline()`: increments synthetic beam counter, ticks CIA-B timer

### `src/recomp_main.c`

- Parses `argv`: `chip_ram_dump.bin disk1 [disk2] [disk3]`
- Calls `hw_init()`, `rt_init()`
- Sets initial `M68KCtx`: `A[7]=0x07FFF0`, `A[5]=0x076000`
- Calls `gfn_003000(ctx)` — should not return
- Defines `const int g_fn_count = GAME_FN_COUNT`

### `src/generated/game.c` (auto-generated)

- Forward declarations for all `gfn_XXXXXX` functions
- Function bodies with goto labels (`L_XXXXXX`) for branch targets
- `const GameFnEntry g_fn_table[GAME_FN_COUNT]` dispatch table
- `const int g_fn_count = GAME_FN_COUNT`
